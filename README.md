## 👀 서비스 소개
* 서비스명:  BPS(업무관리프로그램)
* 서비스설명: 프리랜서들을 위한 업무관리 프로그램
<br>

## 📅 프로젝트 기간

## ⭐ 주요 기능
* GANTT CHART
* CALENDAR
* KANBAN BOARD
* WORKSPACE
